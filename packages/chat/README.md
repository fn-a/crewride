### ai-elements 使用方式

[ai-elements](https://elements.ai-sdk.dev/) 是 Vercel 官方的 AI 组件库，基于 shadcn/ui 构建。

```bash
# 安装 ai-elements CLI
npx ai-elements@latest add message conversation prompt-input

# 组件默认会安装到 components/ai-elements/ 目录下
```

### shadcn-ui 使用方式

```bash
# 使用 shadcn CLI 添加组件（自动安装依赖）
npx shadcn@latest add tabs

# 组件默认会生成到 components/ui/tabs.tsx
```
